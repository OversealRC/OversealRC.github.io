Points reminder

Tied points are decided by distance run.

  1. 1.You get 1 point for going for a run. Simple ↩
  2. 2.Bonus point for each run in addition to last phase ↩
  3. 3.Bonus point for each full 5km run ↩
  4. 4.Bonus point for each achievement (Up to 2 per run/6 per phase) ↩
  5. 5.3 Bonus points for each event (more than 20 runners/tagged as a race) ↩
  6. 6.Weekly bonus: 1 point for each 30% you run more distance than last phase (max 3) ↩
  7. 7.Weekly bonus: 1 point for each 30% you run longer than last phase (max 3) ↩
  8. 8.Weekly bonus: 1 point for each 30% you climb higher than last phase (max 2) ↩
  9. 9.Weekly bonus: 1 point for each 75% more kudos you got than to last phase (max 3) ↩
  10. 10.Weekly bonus: 1 point for each 5% you run faster than last phase (max 2) ↩

About

Founded in 2015, we are a friendly running group catering for all levels.

What started as a post on Facebook asking if anyone would be interested in meeting to run or jog on a Sunday morning, is now a friendly, community spirited running group based in the heart of The National Forest.
